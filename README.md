# Code Overview

- `ShoppingApp/Entities` . This project includes database entities.
    * `ShoppingApp/Entities/Models` . This folder contains Entity Models.
    * `ShoppingApp/Entities/Concrete` . This folder contains Abstarct class and IEntity.
    * `ShoppingApp/Entities/Dto` . This folder contains Data transfer function.
- `ShoppingApp/Core` . This project includes Generic Repository , Service. And Utilities.
- `ShoppingApp/Data` . This project includes Data access layer , UnitOfWork Pattern and entity repositories and their implement class.
- `ShoppingApp/Services` . This project includes project's services.
- `ShoppingApp/WebMVC` . This project contains MVC project.
    * `Areas/Admin` . This folder is admin layer.
    * `AutoMapper` . This folder contains automapper profiles.
    * `Helpers`. This folder contains helper class for controller.
